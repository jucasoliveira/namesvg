A SVG project to programatically build SVG avatars

## Instalation

This project use bun.js

```bash
bun install
```

## Run the project

```bash
bun run dev
```

## To test the project

`http://localhost:3500/imagename`

## Structure

This proejct works on Vite-ssr, to know more about the structure visit: 

`https://vite-plugin-ssr.com/`
